sentiment_analysis module¶

sentiment_analysis.sentiment_analysis(sentence)¶

Analyzes the sentiment of a given sentence using TextBlob and determines its polarity as positive, neutral, or negative.

Parameters:

sentence (str) – The sentence whose sentiment is to be analyzed.

Returns:

A value indicating the sentiment of the sentence: 0 for neutral, -1 for negative, and 1 for positive.

Return type:

int
